FizzBash
===================

Alternative FizzBuzz written by Bash.
This echos 'Bash', instead of 'Buzz'. 

## example

i=1; while [ $i -le 20 ];do ./fizzbash.bash $i; i=$(($i+1)) ;done

Result:

    1
    2
    Fizz
    4
    Bash
    Fizz
    7
    8
    Fizz
    Bash
    11
    Fizz
    13
    14
    FizzBash
    16
    17
    Fizz
    19
    Bash
